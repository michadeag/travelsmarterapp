What is Cents Per Point (CPP)?

CPP is the value you get from each mile or point when redeeming. Higher is better!

  • CPP Calculation: (Cash Equivalent Price ÷ Miles Required) × 100
  • Example: €1,200 redemption ÷ 50,000 miles = 2.4¢ per point
  • Good Value: 1.5¢+ per point is excellent
  • Comparison: Shows which airlines offer best value on same route
